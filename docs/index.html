NightForge

A native, privacy-first desktop app for managing, running, and analyzing deep-learning experiments.

Download View on GitHub

Features

Everything you need to manage your ML workflow — no cloud required.

📊

Project Management

Create and organize ML projects with dataset, model, and training configurations in one place.

🧪

Experiment Tracking

View runs, metrics, and training history in a structured, sortable table.

📈

Charts & Visualization

Interactive loss curves, accuracy plots, and custom metric charts rendered in real time.

🧠

Model Interpretation

Built-in interpretability tools to understand what your model is learning.

🔬

Netron Integration

Visualize neural network architectures directly inside the app.

💻

Integrated Terminal

Full PTY terminal with SSH support for remote training servers — no context switching.

🔒

100% Private

No telemetry, no tracking, no cloud. All your data is stored locally on your machine.

⚡

Lightweight & Fast

Built with Tauri + Rust for a tiny footprint and native performance. No Electron bloat.

🌐

Cross-Platform

Runs natively on macOS (Intel & Apple Silicon), Windows, and Linux.
